⚙️ Configurazione Gemini AI

🚀 Come ottenere la API Key GRATUITA

  1. Vai su Google AI Studio
  2. Accedi con il tuo account Google
  3. Clicca su "Get API Key" o "Create API Key"
  4. Seleziona "Create API key in new project" (o usa un progetto esistente)
  5. Copia la chiave (inizia con AIza...)
  6. Incollala qui sotto

✅ Piano gratuito: 15 richieste/minuto, 1 milione di token/giorno
⚡ Velocità: 2-5 secondi per resoconto (vs 90+ con Ollama locale)
🔒 Privacy: La chiave resta solo sul tuo browser (localStorage)

La chiave viene salvata solo nel tuo browser, non viene inviata a nessun server esterno.

💡 Gemini 2.5 Flash è il più veloce (2-3 secondi) mantenendo alta qualità
