SINGULAR · is / a
PLURAL · are / any / -s
🎵 "Is there a sofa? Yes, there is!" Una cosa = is / a (verde). Tante cose = are / any / -s (arancione). Clicca una riga per evidenziarla.
VERSE 1 · SOFA
Is there a sofa? Yes, there is!
Are there any sofas? Yes, there are.
Is there a sofa? No, there isn't.
Are there any sofas? No, there aren't.
CHORUS
Is there? Yes, there is.
Are there? Yes, there are.
Is there? No, there isn't.
Are there? No, there aren't.
VERSE 2 · BED
Is there a bed? Yes, there is!
Are there any beds? Yes, there are.
Is there a bed? No, there isn't.
Are there any beds? No, there aren't.
VERSE 3 · CHAIR
Is there a chair? Yes, there is!
Are there any chairs? Yes, there are.
Is there a chair? No, there isn't.
Are there any chairs? No, there aren't.
